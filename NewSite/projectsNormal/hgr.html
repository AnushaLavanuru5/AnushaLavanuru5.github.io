Hand Gesture Recognition System

Overview

Purpose: Developed a machine learning-based system to recognize static hand gestures, enhancing human-computer interaction (HCI) for applications in automation, gaming, and accessibility.

Tech Stack

  • Programming Languages: Python
  • Libraries and Tools: OpenCV, TensorFlow, Keras,
  • Dataset: NUS Hand Posture Dataset

Key Features

  • Robust Feature Extraction: Implemented Histogram of Oriented Gradients (HOG) and Fibonacci Weighted Neighborhood Patterns (FWNP) for accurate gesture recognition.
  • Real-Time Gesture Detection: Designed a pipeline for real-time gesture detection using webcam input, leveraging advanced image processing techniques.
  • Enhanced System Performance: Improved preprocessing methods such as segmentation, noise reduction, and edge detection for gesture isolation.
  • Adaptability: Incorporated flexibility to recognize gestures under varying conditions, including changes in lighting, background complexity, and scale.

Project Highlights

  • Touchless Control: Enabled control for electronic devices and home automation systems without physical interaction.
  • Gaming and Entertainment: Enhanced interaction for immersive gameplay experiences.
  • Accessibility: Enabled sign language translation to bridge communication barriers.
  • Defense Applications: Developed secure, gesture-based control systems for military and defense use.

Outcome

  • High Recognition Accuracy: Achieved reliable classification across diverse conditions, ensuring robust performance.
  • Enhanced User Experience: Improved HCI systems with responsive and scalable gesture recognition technology.
  • Real-World Applications: Showcased the potential of ML-driven solutions for accessibility, automation, and beyond.
