Image Enhancement using GANs

Overview

Developed a machine learning model using Generative Adversarial Networks (GANs) to enhance the visual quality of images by improving sharpness, contrast, and brightness.

Addressed challenges in processing low-resolution and noisy images, creating high-quality outputs suitable for real-world applications.

Tech Stack

  • Programming Languages: Python
  • Libraries: OpenCV, TensorFlow, Keras
  • Dataset:MIT-Adobe 5K Dataset (5,000 images)

Key Features

  • Optimized Image Enhancement: Designed a U-Net generator integrated with global features to optimize local and global characteristics for accurate image enhancement.
  • Improved Training Stability: Utilized Wasserstein GAN (WGAN) for faster convergence and enhanced training stability through an adaptive weighting scheme.
  • Automated Preprocessing: Automated steps like segmentation, noise reduction, and edge detection for enhanced input clarity.

Project Highlights

  • Improved Metrics: Achieved significant improvement in PSNR (Peak Signal-to-Noise Ratio) compared to other models, ensuring superior image quality.
  • Enhanced Training Process: Utilized cycle consistency with two-way GANs for bidirectional image mapping and consistency checks.
  • Validated Results: Verified robustness across diverse image conditions using SSIM (Structural Similarity Index).

Applications

  • Photo Enhancement: Improving images for professional use, social media, and creative industries.
  • Medical Imaging: Providing detailed visuals for structural cell and tissue analysis.
  • Underwater Photography: Enhancing clarity in low-light and grayscale environments.
  • Video Processing: Adapting the model for real-time video enhancement tasks.

Outcome

  • Scalable and Efficient Pipeline: Delivered a system capable of enhancing images in real-time.
  • Diverse Applications: Demonstrated potential for use in media, healthcare, and environmental imaging.
